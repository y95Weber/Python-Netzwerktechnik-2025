# PingPong Projekt (UDP)

Dieses Projekt besteht aus zwei Python-Programmen, die eine einfache Ping-Pong-Kommunikation über UDP realisieren.  
Der Client (Ping) sendet eine Zahl **n**, und der Server (Pong) antwortet mit **n + 1**.

---

## Starten

1. `pong_udp.py` starten  
Der UDP-Server läuft und wartet auf Nachrichten.

2. `ping_udp.py` starten  
Der Client verbindet sich nicht direkt, sondern sendet UDP-Pakete an den Server.

3. Zahl eingeben  
Der Server antwortet mit **n+1**

Beenden mit:  quit
